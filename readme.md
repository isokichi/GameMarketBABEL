# キーを押して動画と音楽を再生

このアプリケーションは、キーボードの特定のキーを押すことで動画と音楽を再生・制御します。

## 操作方法

*   **'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p', 'a', 's' キー**:
    対応する動画の待機画面に移動します。
*   **スペースキー**:
    動画の再生/一時停止を切り替えます。待機画面が表示されている場合は動画を再生し、動画が再生中の場合は一時停止/再生を切り替えます。
*   **'F' キー**:
    フルスクリーンモードのオン/オフを切り替えます。
*   **'L' キー**:
    結果発表画面の表示/非表示を切り替えます。
